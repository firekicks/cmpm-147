Experiment 4 - Infinite Worlds - Fish Rumble

Description

This world made up of tiles and animated fish. It generates a grid of tiles with varying colors, simulating water. The tiles can shake and change color when clicked, with neighboring tiles also being affected. Meanwhile, fish swim around the screen, occasionally spawning more fish over time, adding dynamic movement to the scene. The overall effect is a lively, interactive visual display.

Technical

This code defines a visual simulation of a world made up of tiles and animated fish. It initializes an array of fish objects and, through a world key, sets a seed to generate consistent noise and random values. Clicking a tile makes it shake and change color, also affecting neighboring tiles with reduced shaking strength. The code manages these states through several global structures. Before each frame is drawn, fish states are updated, and new fish spawn periodically. Tiles are drawn with potential shaking effects, and fish are drawn as ellipses with oscillating fins and circular eyes. The result is a dynamic and interactive display.
